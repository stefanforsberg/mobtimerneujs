# Mob timer

## Getting started

1. `npm install`
1. `npm run update`

## Development

Start the application in development mode:

`npm start`

If you prefer running the neu CLI globally:

1. `npm install -g @neutralinojs/neu`
1. `neu run`

## Release

`npm run build` (or `neu build --release` via the neu CLI) to create binaries.
